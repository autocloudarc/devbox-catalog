# DevCenter Catalog for DevBox

This repository serves as a quick-start catalog that could be attached to the Dev Center resource in Microsoft Azure, for use in Microsoft Dev Box. You may choose to attach this repository as a catalog that provides sample teamplates to enable experiences such as customizing a Dev Box using Tasks.
